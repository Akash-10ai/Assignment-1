# Assignment-02: String Utility Toolkit

## Objective

The objective of this assignment is to learn how to perform different string operations in C using arrays, loops, and functions. It also helps in understanding how strings are stored and manipulated.

---

## Problem Statement

Write a menu-driven C program that performs different string operations such as finding the length of a string, reversing the string, checking whether it is a palindrome, counting vowels, digits and spaces, and displaying the frequency of each character.

---

## Algorithm

1. Start the program.
2. Enter a string.
3. Display the menu.
4. Read the user's choice.
5. Perform the selected operation:
   - Find string length
   - Reverse the string
   - Check palindrome
   - Count vowels, digits and spaces
   - Display character frequency
6. Repeat the menu until the user selects Exit.
7. Stop the program.

---

## Time Complexity

| Operation | Time Complexity |
|-----------|-----------------|
| Find Length | O(n) |
| Reverse String | O(n) |
| Palindrome Check | O(n) |
| Count Vowels, Digits & Spaces | O(n) |
| Character Frequency | O(n) |

---

## Space Complexity

- Overall Space Complexity: **O(n)**
- Auxiliary Space: **O(1)**
